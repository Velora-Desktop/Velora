# Velora AW0.07

Первая цельная публичная альфа персональной медиатеки для Windows. Velora объединяет игры, фильмы, сериалы и программы, хранит личные данные локально и использует отдельно обновляемый официальный каталог.

Текущий официальный каталог: `AW0.0711` (42 тестовые карточки).

## Возможности AW0.07

- единый каталог четырёх типов контента;
- личные оценки по критериям, статусы, избранное и хронология;
- локальный профиль и отдельная пользовательская SQLite-база;
- Quick View и канонические страницы объектов по стабильному `catalog_id`;
- личная статистика, глобальный поиск и фильтрация;
- безопасное скрытие контента 18+;
- единая тёмная визуальная система и официальный набор SVG-иконок.

## Запуск

```powershell
python -m pip install -r requirements.txt
python main.py
```
