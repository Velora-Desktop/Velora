# Velora assets

- `icons/` — reusable monochrome SVG icons.
- `logos/` — Velora brand marks.
- `covers/` — bundled official cover assets when packaging is introduced.
- `placeholders/` — fallback art for missing media.
- `fonts/` — locally bundled fonts and their licenses.
- `styles/` — non-Python style resources.
- `ui/` — composite UI artwork that is not a semantic icon.

Do not store user-selected covers or avatars here.
