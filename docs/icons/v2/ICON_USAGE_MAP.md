# Карта использования

## Velora
- Sidebar: `media_types`, `statuses`, `ui`
- Quick View: `metrics`, `details`
- Game Detail: `details`
- Statistics: `statistics`, `trend_up`, `activity`, `progress`
- Settings/Data: `diagnostics`
- Dialogs: `info`, `warning`, `error`, `success`

## Velora Studio
- Toolbar: `ui`
- Structure tree: `category`, `subcategory`, `catalog`
- Card editor: `details`
- Database tools: `diagnostics`
- Backup/import/export: `backup`, `restore`, `import`, `export`

## Не использовать как декор
`warning`, `error`, `repair`, `database_check` должны иметь строгую семантику.
