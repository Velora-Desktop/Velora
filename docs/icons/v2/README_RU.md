# Velora Icon Pack v2 — Missing Icons

Второй пакет закрывает недостающие **служебные, статусные, метрические и диагностические иконки**
для Velora AW0.08 и Velora Studio.

## Особенности

- 24×24 SVG;
- единый line-style;
- `stroke="currentColor"` — иконки можно перекрашивать через Qt;
- прозрачный фон;
- без брендовых логотипов;
- подходит для Full HD, 2K и 4K;
- не заменяет Velora Icon Pack v1, а дополняет его.

## Папки

- `ui` — основные действия интерфейса;
- `media_types` — игры, фильмы, сериалы, программы и профиль;
- `statuses` — статусы разных типов контента;
- `metrics` — время, просмотры, сезон, эпизод, оценки;
- `details` — метаданные карточки и системные требования;
- `diagnostics` — backup, базы, логи и диагностика.

## Использование в PySide6

```python
from PySide6.QtGui import QIcon

icon = QIcon("assets/icons/v2/ui/search.svg")
```

Рекомендуется подключать через централизованный `IconRegistry`, а не через прямые пути.

## Размеры

- таблицы и мелкие кнопки: 14–16 px;
- sidebar / toolbar: 18 px;
- диалоги и секции: 20–24 px;
- крупные пустые состояния: 32–48 px.

## Важно

Брендовые платформы и сервисы остаются в Icon Pack v1.
